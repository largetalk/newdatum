===============================
股市日记2015.2.2
==============================

今天股市又大跌，亏损严重，上周操作太多，果然应了那句话，多做多错，手中现金几乎已经没有，导致现在很被动．
上周错误点主要有三：
１．看好银行股，买时却不看价，没有耐心，在最近的高点（上证３３００左右）买入，应该细心等待，在股市里，机会是永远存在的，在京东方等股票上已经吃过这样的亏，还是没记住
２．国元３６卖出后３３买入，沾沾自喜，违背了先买后卖的原则且没有注意大盘走势，买在了半山腰
３．妄图做Ｔ又太贪心，７３买入平安到７４．５了还不卖出，持股过夜风险大，现在跌成这样吃了一个教训，该做Ｔ就应该做Ｔ，赢利一点就可以卖出，不要太贪心，并且现在看来做Ｔ风险很大，如果拿住现金，现在补仓，比做Ｔ安全．

股市风险确实很大，只有极少有智慧，有耐心，能坚持原则的人才能获得最后的收益，现在的教训要好好反省，多做功课，少看盘，少操作，加油！
